**TLDR** 配置环境，写自己的docker，跑自己的app并且部署到云上


# 安装

Follow [docker doc](https://docs.docker.com/engine/install/ubuntu/) 安装 docker engine。

在ubuntu里启动 docker daemon

```bash
sudo dockerd
```

# 第一个docker run
```bash
sudo docker run hello-world
```
